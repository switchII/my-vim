## 说明  
自己常用的gvim配置，方便个人使用.

## 安装
    1.安装方法为在终端输入如下命令,先下载 vundel来管理插件 
    `git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle`
    2.把vim文件夹和gvimrc文件修改名称为.vim 和.gvimrc
    3.在gvim中执行PluginInstall命令，安装完插件就差不多的了
